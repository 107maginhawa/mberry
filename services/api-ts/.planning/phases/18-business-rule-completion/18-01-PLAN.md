<action>
Find the nomination creation handler in the elections module. Add these eligibility checks before creating a nominee:

1. **Active membership check**: Verify the nominee has `status: 'active'` in the organization's membership table at nomination time.
```typescript
const membership = await db.select().from(memberships)
  .where(and(
    eq(memberships.personId, nomineePersonId),
    eq(memberships.organizationId, orgId),
    eq(memberships.status, 'active')
  )).limit(1);
if (!membership.length) {
  throw new BusinessLogicError('Nominee must be an active member of the organization', 'NOMINEE_NOT_ACTIVE');
}
```

2. **Minimum tenure check**: Verify the nominee has been a member for at least 6 months (use membership `createdAt` or `joinedAt` field).
```typescript
const sixMonthsAgo = new Date();
sixMonthsAgo.setMonth(sixMonthsAgo.getMonth() - 6);
if (new Date(membership[0].createdAt) > sixMonthsAgo) {
  throw new BusinessLogicError('Nominee must have been a member for at least 6 months', 'NOMINEE_INSUFFICIENT_TENURE');
}
```

3. **No suspension check**: Verify nominee is not suspended in ANY organization.
```typescript
const suspended = await db.select().from(memberships)
  .where(and(
    eq(memberships.personId, nomineePersonId),
    eq(memberships.status, 'suspended')
  )).limit(1);
if (suspended.length) {
  throw new BusinessLogicError('Nominee cannot be suspended in any organization', 'NOMINEE_SUSPENDED');
}
```

Import BusinessLogicError from `@/core/errors`. Adapt field names to match actual schema columns.
</action>
