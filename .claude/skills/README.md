# Claude Code Skills

This directory contains 21 project-specific skills for Claude Code — Anthropic's AI coding assistant. Each skill automates a development workflow (e.g., creating TypeSpec definitions, running tests, generating migrations).

## For developers NOT using Claude Code

You can safely ignore this directory. These files don't affect the build, tests, or runtime behavior. They're automation scripts for AI-assisted development.

## For developers using Claude Code

Skills appear as `/skill-name` commands in Claude Code sessions. Type `/typespec` to author API definitions, `/handler` to implement business logic, `/test-api` to run tests, etc. See each skill's `SKILL.md` for details.
