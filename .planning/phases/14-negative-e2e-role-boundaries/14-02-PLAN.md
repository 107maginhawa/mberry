```typescript
export const SEED_IDOR_EMAIL = process.env.SEED_IDOR_EMAIL ?? 'idor-officer@memberry.ph'
```

**3 Cross-org isolation tests:**

1. **Org B officer cannot view Org A roster**
   - Sign in as `idor-officer@memberry.ph` (Org B president)
   - Attempt to fetch Org A roster via API: GET `/association/member/roster-members?organizationId={ORG_A_ID}`
   - Expect 403

2. **Org B officer cannot view Org A dues dashboard**
   - Sign in as IDOR officer
   - Attempt GET `/dues/dashboard?organizationId={ORG_A_ID}`
   - Expect 403

3. **Org B officer cannot manage Org A events**
   - Sign in as IDOR officer
   - Attempt POST `/association/operations/events` with `organizationId: ORG_A_ID`
   - Expect 403

Use constants:
```typescript
const ORG_A_ID = 'ed8e3a96-8126-4341-be42-e6eb7940c562' // pda-metro-manila
```

Pattern:
```typescript
test('org B officer cannot view org A roster', async ({ page }) => {
  await signIn(page, SEED_IDOR_EMAIL, TEST_PASSWORD)
  const status = await page.evaluate(async ({ apiBase, orgAId }) => {
    const res = await fetch(`${apiBase}/association/member/roster-members?organizationId=${orgAId}`)
    return res.status
  }, { apiBase: API_BASE, orgAId: ORG_A_ID })
  expect(status).toBeGreaterThanOrEqual(400)
})
```
