    Implementation for export:
    - Use `useQuery` with `exportMyDataOptions()` and `enabled: false` to create a lazy query
    - On click: `refetch()`, then create a Blob from the JSON data, trigger browser download with filename `my-data-YYYY-MM-DD.json`
    - Use `toast.success('Data export downloaded')` on success, `toast.error(...)` on failure
    - Pattern from RESEARCH.md:
      ```tsx
      const { refetch: fetchExport, isFetching: isExporting } = useQuery({
        ...exportMyDataOptions(),
        enabled: false,
      });
      const handleExport = async () => {
        const result = await fetchExport();
        if (!result.data) { toast.error('Failed to export data'); return; }
        const blob = new Blob([JSON.stringify(result.data, null, 2)], { type: 'application/json' });
        const url = URL.createObjectURL(blob);
        const a = document.createElement('a');
        a.href = url;
        a.download = `my-data-${new Date().toISOString().slice(0, 10)}.json`;
        a.click();
        URL.revokeObjectURL(url);
        toast.success('Data export downloaded');
      };
      ```
