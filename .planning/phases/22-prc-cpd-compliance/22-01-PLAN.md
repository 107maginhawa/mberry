   - `prcAccreditationNumber: varchar('prc_accreditation_number', { length: 100 })`
   - `accreditedProviderId: uuid('accredited_provider_id')`
   Both nullable (no `.notNull()`) — existing trainings have no provider. Do NOT add a DB-level FK constraint (cross-handler-directory import complexity; validate in handler instead).

2. **credits.schema.ts** — Add two new pgEnums before the `creditEntries` table:
   - `export const cpdCategoryEnum = pgEnum('credit_cpd_category', ['General', 'Major', 'Self-Directed']);`
   - `export const verificationStatusEnum = pgEnum('credit_verification_status', ['pending', 'verified', 'rejected']);`
   Use namespaced enum names (`credit_cpd_category`, `credit_verification_status`) to avoid PostgreSQL global enum name collisions.
   Add three columns to `creditEntries` table after `supportingDocumentId`:
   - `category: cpdCategoryEnum('category')`  (nullable — existing entries have no category)
   - `approvalCode: varchar('approval_code', { length: 100 })`  (nullable)
   - `verificationStatus: verificationStatusEnum('verification_status').notNull().default('pending')`
   Export the new enum types and update the `CreditEntry`/`NewCreditEntry` type exports (they auto-infer from table).

3. **accredited-provider.schema.ts** — Create new file at `services/api-ts/src/handlers/training/repos/accredited-provider.schema.ts`:
   ```typescript
   import { pgTable, varchar, uuid, pgEnum, index, timestamp } from 'drizzle-orm/pg-core';
   import { baseEntityFields } from '@/core/database.schema';

   export const providerStatusEnum = pgEnum('accredited_provider_status', ['active', 'suspended', 'expired']);

   export const accreditedProviders = pgTable('accredited_provider', {
     ...baseEntityFields,
     organizationId: uuid('organization_id').notNull(),
     name: varchar('name', { length: 300 }).notNull(),
     accreditationNumber: varchar('accreditation_number', { length: 100 }).notNull(),
     status: providerStatusEnum('status').notNull().default('active'),
     expiryDate: timestamp('expiry_date'),
   }, (table) => [
     index('idx_accredited_provider_org').on(table.organizationId),
     index('idx_accredited_provider_status').on(table.status),
   ]);

   export type AccreditedProvider = typeof accreditedProviders.$inferSelect;
   export type NewAccreditedProvider = typeof accreditedProviders.$inferInsert;
   ```
