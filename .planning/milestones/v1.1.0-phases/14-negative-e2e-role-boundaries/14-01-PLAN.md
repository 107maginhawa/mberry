**Section 1: Member cannot access officer routes (6 tests)**

Sign in as `SEED_MEMBER_EMAIL` (`member@memberry.ph`). For each officer route below, navigate and verify either:
- Redirect to non-officer page (URL does not contain `/officer/`), OR
- API call returns 403 (via `page.evaluate` fetch)

Routes to test:
1. `/org/{orgId}/officer` — officer dashboard
2. `/org/{orgId}/officer/members` — member management
3. `/org/{orgId}/officer/finances` — dues/payments
4. `/org/{orgId}/officer/events` — event management
5. `/org/{orgId}/officer/communications` — announcements
6. `/org/{orgId}/officer/settings` — org settings

Use ORG_ID constant `ed8e3a96-8126-4341-be42-e6eb7940c562` (from seed).

Pattern for each test:
```typescript
test('member cannot access officer dashboard', async ({ page }) => {
  await signIn(page, SEED_MEMBER_EMAIL, TEST_PASSWORD)
  await page.goto(`/org/${ORG_ID}/officer`)
  await page.waitForLoadState('networkidle')
  await page.waitForTimeout(2000)
  // Member should be redirected away or see error
  const url = page.url()
  const hasOfficerAccess = url.includes('/officer') && !url.includes('/auth/')
  if (hasOfficerAccess) {
    // If frontend doesn't block, verify API returns 403
    const res = await page.evaluate(async (apiBase) => {
      const r = await fetch(`${apiBase}/association/operations/dashboard?organizationId=ed8e3a96-8126-4341-be42-e6eb7940c562`)
      return r.status
    }, API_BASE)
    expect(res).toBeGreaterThanOrEqual(400)
  }
})
```

```typescript
test('treasurer cannot create events', async ({ page }) => {
  await signIn(page, SEED_TREASURER_EMAIL, TEST_PASSWORD)
  const status = await page.evaluate(async ({ apiBase, orgId }) => {
    const res = await fetch(`${apiBase}/association/operations/events`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        organizationId: orgId,
        title: 'Unauthorized event',
        startDate: '2026-06-01T09:00:00Z',
        endDate: '2026-06-01T17:00:00Z',
      }),
    })
    return res.status
  }, { apiBase: API_BASE, orgId: ORG_ID })
  expect(status).toBeGreaterThanOrEqual(400)
})
```
